## How to Participate

1.  **Clone** this repository.
2.  **Create your own branch**.
3.  Navigate to the folder of the active challenge (`/challenges/weekX`, etc.).
4.  Use the folder of your chosen language (e.g., `/challenges/weekX/java`).
5.  Place your solution into the solution file (e.g., `/challenges/weekX/java/Solution.class`).
6.  **Commit** your solution **to your branch** and **Submit a Pull Request (PR)** to the main branch to execute the pipeline with the tests.
